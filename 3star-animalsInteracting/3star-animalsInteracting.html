Instructions

This exercise builds on the 1 and 2 star exercises, so you will need to complete those before beginning this one.

Once again, your task is to extend the Animal class you wrote in the previous exercise with new functionality described below. Start by copying the code and assets from the last exercise.

Modify the Animal class so that each animal can have its own sound. You should also have a public method that can be called to play the sound.

Add input controls to the sketch that will enable the user to select a single animal to be "active". Modify your keyPressed() implementation so that only the active animal moves. Add new functionality so that when the user clicks on the canvas, the active animal moves to the mouse location.

Finally, when one animal gets too close (you decide what that means!) to another, play the other animal's sound. For example, if animal A gets too close to animal B, animal B's sound should play. Think carefully about how to design this functionality. You may need to add new methods to your Animal class. Some recommendations are provided in the hint.

If you need a reminder on working with audio, see the Sound documentation, especially the section on working with sound files. The library is already linked.

The DOM section of the p5.js reference provides guidance on working with input controls. Buttons, a radio group, or a select are the recommended controls, of which buttons are the easiest to work with. Styling radio groups can be quite finnicky!

When working out the distance between two animals, you may find the p5.js dist() function helpful. In addition to deciding how close is too close, you will have to decide where and when to calculate the distance between animals. In terms of where, you can either add a method to the Animal class or make a new function in the main sketch. In terms of when to check the distance between animals, aim for efficiency. You should not need to check unless an animal has moved (i.e., don't check every time draw() runs).

If your Animal attributes are private, you will almost certainly need to add getter functions (e.g. getX() and getY()) to complete this exercise.
